# Three-Tier DevOps Project 🚀

## Architecture
Frontend (React) → Backend (Node.js) → MongoDB

## Technologies Used
- React
- Node.js
- MongoDB
- Docker
- Docker Compose

## Features
- Add Tasks
- Delete Tasks
- MongoDB Data Storage
- Dockerized Application

## Run Project

```bash
docker-compose up --build -d

Frontend:
